Extensions

See how you can improve Buefy with other packages

Buefy has no dependencies aside from Bulma. But sometimes we need to add a very specific funcionality to our application.

These extensions are some examples of how you can extend some of the components using external libraries. And by no means is the only ones "supported", you can play with it and add basically anything you may want.

Also, don't forget to add a Pull Request if you've made one :)

All examples shown here are not built-in Buefy, you have to install dependencies for each one.
